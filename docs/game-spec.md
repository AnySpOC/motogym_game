# Moto Gymkhana Game Spec

## Concept

モトジムカーナを題材にしたリアル志向のタイムアタックゲーム。
小さな舗装エリアに置かれたパイロン課題を、指定方向に旋回しながら最短タイムで走る。

## Current Prototype

Three.js版は `web-prototype/` に格納する。
Unity版はこの仕様を基準に段階的に移植する。

## Core Rules

- 赤パイロン: 右回り
- 緑パイロン: 左回り
- 黄/ライン: ゲートまたは進行方向補助
- パイロン接触: `+1秒`
- コースアウト: `+3秒`
- ミスコース: ゲームオーバー後に自動再開
- 転倒: ゲームオーバー後に自動再開
- ゴール: 全ゲートを順番に通過した時点のタイム + ペナルティ

## Controls

- `W`: 加速
- `S`: 減速
- `A`: 左バンク
- `D`: 右バンク
- `Q`: リーン左
- `E`: リーン右
- `Space`: リアブレーキ
- `R`: 復帰
- マウス/タッチドラッグ: 視点回転

## Physics Direction

リアル志向にするため、操作で直接曲げるのではなく、以下の流れにする。

1. 入力から目標バンク角を作る
2. バイクがリーンレートを持って倒れる
3. バンク角からヨーレートを計算する
4. 低速時のみ前輪舵角を補助的に効かせる
5. リアブレーキや大きなバンクでリアスライドを発生させる

簡易式:

```text
yawRate = g * tan(bankAngle) / speed
```

## Unity Milestone

1. 空のUnity 3Dプロジェクトを `unity/` に作る
2. 平面コース、赤/緑パイロン、スタート/ゴールを配置
3. バイクを仮モデルで走行可能にする
4. タイマーとペナルティを実装
5. WebプロトタイプのコースをUnityへ移植
6. Blenderモデルに差し替える
